# 🧠 Zenith — Painel Web do Psicólogo


Sistema web desenvolvido como parte do TCC, integrante do projeto **Zenith**, uma plataforma de gerenciamento terapêutico.

O objetivo deste sistema web é facilitar o gerenciamento de pacientes, sessões e acompanhamento clínico de forma organizada, segura e intuitiva.


---

## 📌 Sobre o Projeto


O **Zenith** é um sistema criado para conectar o paciente ao profissional, além de auxiliar profissionais da psicologia na organização do atendimento clínico, centralizando informações importantes em um único ambiente digital.

Este repositório contém o **painel web do psicólogo**, onde o profissional pode administrar sua rotina de atendimentos.


---

## ✨ Funcionalidades


| Área | Descrição |
|------|-----------|
| 👤 Pacientes | Cadastro, edição e gerenciamento de pacientes |
| 📅 Sessões | Agendamento, remarcação e controle de atendimentos |
| 🧾 Anotações | Registro de observações clínicas |
| 📊 Agenda | Visualização dos atendimentos do dia/semana |
| 💰 Financeiro | Controle de pagamentos *(em desenvolvimento)* |
| 🔐 Autenticação | Login seguro do profissional |
